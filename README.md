# ⌨️ karabiner

custom global keyboard shortcuts for macos.

![example screenshot](/screenshot.png)

needs [karabiner elements](https://karabiner-elements.pqrs.org).

**caps → hyper / escape**
caps lock becomes a hyper key (`right_shift` + `right_command` + `right_control` + `right_option`) when held, and sends `escape` when tapped on its own. hyper is great for global shortcuts that won't clash with anything else.

**hyper + wasd → arrows**
hold caps and use `wasd` as arrow keys.

## install

clone into `~/.config/karabiner`:

```bash
git clone git@github.com:aldosch/karabiner.git ~/.config/karabiner
```
